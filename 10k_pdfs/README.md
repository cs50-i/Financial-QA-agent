
# 10-K PDF Directory

To use this system with your own 10-K PDF filings:

1. Place your PDF files in this directory
2. Name them clearly (e.g., "NVDA_2024_10K.pdf", "AAPL_2023_10K.pdf")
3. The system will automatically extract company names and years from filenames

Supported filename patterns:
- nvidia_2024_10k.pdf
- AAPL_2023_10K.pdf
- msft-2022-10k.pdf

The system will process all PDFs and make them searchable!
